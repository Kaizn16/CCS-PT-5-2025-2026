Module Overview

Welcome to the fundamentals of C programming, flowcharting, and pseudocode, as well as an introduction to the core concepts of Object-Oriented Programming (OOP).

Course Requirements

Attendance: 20%
Quiz & Activity: 20%
Midterm & Final Exam: 60%

Topics Covered

1. Getting Started with C

A brief history of C, its importance, and how to write and compile your first "Hello, World!" program.

2. Variables & Data Types

Understanding how to store data using variables and the different types of data (int, float, char, etc.).

3. Input & Output

Using `printf()` and `scanf()` to interact with the user and display information on the screen.

4. Operators

Learning about arithmetic, relational, logical, and other operators to perform operations on data.

5. Flow Control

Using `if...else`, `switch`, `for`, and `while` statements to control the flow of your program.

6. Functions

Creating modular and reusable code by defining and calling functions.

7. Flowcharting & Pseudocode

Learning to plan and design your program's logic before writing code using visual flowcharts and plain-language pseudocode.

8. Introduction to OOP

An overview of Object-Oriented Programming (OOP) concepts like objects and classes. We'll also cover the four pillars of OOP:

• Abstraction: Hiding complex implementation details and showing only the essential features.
• Encapsulation: Bundling data and the methods that operate on that data into a single unit.
• Polymorphism: The ability of an object to take on many forms.
• Inheritance: A mechanism where a new class is derived from an existing class, inheriting its properties.
